* Esto debe de hacerse antes de de correr el proyecto, ya que los "node_modules" no reconoce las carpetas *

*** Terminal de Angular ***

1. Carousel
ng add ngx-bootstrap  --component carousel
Esto es para que haga la animación de las fotografias

2. Dropzone
npm i ngx-dropzone
Esto para el espacio para seleccionar las fotografías


** Terminal del servidor **

1. Multer
npm i multer
Esto para la creación de las fotografías
